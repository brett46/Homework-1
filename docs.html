Our plan is to create a mobile-first web app to cater to the statistics needs of the Whitworth Baseball team. We will use a mongoDB back-end on a google cloud compute instance, running a Django-wsgi application through apache's wsgi module.


The application will have a responsive-design and allow the team to take statistics on an ipad, laptop or phone while the game is going on, and be able to access different parts of the site as needed to run automated queries on the database. The django-backend will also run machine learning algorithms on the data to give predictive analysis on the data in real time to assist game decisions.
